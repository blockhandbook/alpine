== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `parts` plugin files to the `/wp-content/plugins/` directory, or upload the parts.zip file through the WordPress plugins screen directly by clicking 'Add New' and selecting the zip file from your computer.
2. Install and active the Gutenberg WordPress plugin.
3. Activate the Parts plugin through the 'Plugins' screen in WordPress.
4. Use the Parts plugin on your pages and posts.
